Interface

Dependencies
------------

Per pip:
```
pip install -r requirements.txt
```

Anonsten gibt es alle Dependencies auch in den Debian/Ubuntu Repos. Gebraucht wird hier Python 3, PyQt5,
Python-Requests und Python-Pillow.
